문제   
[Intro to Conditional Statements](https://www.hackerrank.com/challenges/30-conditional-statements/problem)

~~~
if __name__ == '__main__':
    N = int(input())
    if 2 <= N and N <= 5 and N % 2 == 0:
        print("Not Weird")
    elif 6 <= N and N <= 20 and N % 2 == 0:
        print("Weird")
    elif N % 2 == 1:
        print("Weird")
    elif N % 2 == 0:
        print("Not Weird")
~~~
